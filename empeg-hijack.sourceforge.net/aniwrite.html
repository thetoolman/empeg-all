Empeg aniwrite -- shell tool to write/clear boot animations

This tool can be used to write or clear a custom boot-up animation on the Mk1/Mk2/Mk2a Empeg/RioCar player. This works only with Hijack kernels, not the stock Empeg/Rio player kernel.

Usage is simple. Place the aniwrite executable onto your player, do a chmod 0755 aniwrite to make it executable, and then run it with a raw animation file as the lone argument. Then reboot. To clear (remove) a custom animation, use aniwrite clear
