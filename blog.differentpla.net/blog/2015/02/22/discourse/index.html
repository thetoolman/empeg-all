I installed discourse

2015-02-22 14:41:00 +0000 blogging

I’ve installed Discourse at discourse.differentpla.net as a way of enabling comments on this blog.

I’ve enabled Google, Twitter and Github login, so it should be relatively frictionless to post comments and/or criticism of my blog posts.

The actual installation of Discourse was relatively simple, so I don’t think I’ll bother writing that up.

Oh, apart from configuring DNS for Mailgun. I wrote that up.
