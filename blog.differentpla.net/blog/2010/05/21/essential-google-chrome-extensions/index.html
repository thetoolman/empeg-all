Essential Google Chrome Extensions

2010-05-21 09:56:01 +0000

These are the extensions that I have to install whenever I set up a new copy of Google Chrome.

  • Domain in Title – essential if you’re using anything like KeePass, which recognises windows by their caption. This extension puts the domain name in the Google Chrome title bar. Alternatively, there’s Host in Title or Url in title.
  • Google Mail Checker Plus
  • Google Reader Notifier (by Google)
  • RSS Subscription Extension (by Google)
  • Stack Overflow Notifier

You might be wondering why there’s no AdBlock, FlashBlock or ScriptBlock extensions in the list. It’s mainly because I don’t care.
