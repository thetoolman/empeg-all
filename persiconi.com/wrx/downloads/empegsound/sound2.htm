Your MP3s
How to create great sounding music files.
Before we can go about tweaking your hardware for the best sound possible, we need to ensure that you have good source material.  And the only way to ensure quality MP3s is to make them yourself and that means ripping your own CDs.

Countless programs exist, and opinions vary on which is the best to use, but few people can argue that Exact Audio Copy (EAC) isn't one of the most powerful choices.  Better yet, it's free!  EAC gets its name because it is the only program that has special read modes, known as "Secure Modes". Using these secure modes, every sector read will be double-checked and reread or corrected if necessary. On many drives the extraction is not error free, thus these routines will make sure the track is read correctly.  Once the audio is copied from the CD, we'll use the LAME MP3 encoder to give us high quality Variable Bit Rate (VBR) MP3s.

There are a multitude of setup guides for EAC available on the web, but this should get you running in no time. EAC is a powerful program with many settings that need to be configured. This guide provides you with a "profile" that you load into EAC that sets up all the complex options for you.  After that, you can go in and make changes that best suit you.


 

To set up EAC, perform the following steps:

  1. Download and install Exact Audio Copy V0.9 beta 4 to C:\Program Files\Exact Audo Copy.
  2. For Windows NT, 2000 and XP users: Make sure EAC is NOT running, then download the Nero ASPI Layer DLL. You must place this DLL file in the directory you installed EAC. (C:\Program Files\Exact Audio Copy). NOTE: Windows 95 and 98SE users should skip this step.
  3. Download LAME v3.90.3 (Dibrom's Build) and place the LAME executable in the following path: C:\Program Files\Exact Audio Copy\lame.exe. (LAME is the program that encodes the MP3s).
  4. Download the EAC Profile.CFG and place the profile in the following path: C:\Program Files\Exact Audio Copy\Profiles\
  5. Open Exact Audio Copy and load the EAC Profile you downloaded in Step 4 (EAC > Profiles > Load Profile). This automatically sets up most of the correct settings in EAC.
  6. For the remaining settings, do the following after you have loaded the profile:
    • EAC > EAC Options > Interface tab: Select "Installed external ASPI interface."  If this option is not available, double check where you saved the Nero ASPI Layer DLL file. Screen-shot.
    • EAC > Drive Options > Drive tab:  Insert a music CD into your CD-RW drive, then click the Autodetect Read Command Now button, and click OK.  Next, make sure that the "Secure mode with the following features" option is selected.  Do NOT use "Burst" or "Paranoid" modes.  Screen-shot.
    • EAC > Drive Options > Offset/Speed tab: Skip the read/write offsets. Select the Overread into Lead-In and Lead-Out check box if your drive supports it. In the Speed Selection field, select the speed that your CD-RW reads at. Select the Allow speed reduction during extraction check box. Screen-shot.
    • EAC > Compression Options > External Compression tab: Click the Browse button and locate the lame.exe file (C:\Program Files\Exact Audio Copy\lame.exe). In the Additional command line options field, enter your preferred command line option (--alt-preset standard, extreme, or insane). Screen-shot.
    • EAC > freedb/Database Options > freedb tab: To access the online freedb databases, enter a valid e-mail address, cick the Get active freedb server list button, and select any of the freedb servers.
  7. Save the profile (EAC > Profiles > Save Profile).
  8. Simply insert your first CD and hit the "MP3" button to begin ripping.
  9. After you rip your first CD, check the LOG file and make sure that the Read Mode displays as Secure with NO C2, Accurate Stream, Disable Cache. If your LOG file does not display the correct read mode, try re-loading the .CFG file again. Users who have multiple CD-RW drives will need to load the EAC Profile.CFG file for each CD-RW drive.

Various Artist CDs: If you are ripping a Various Artists CD, be sure to select the Various Artists check box in the main EAC window, and enter the name of the CD in the CD Title field. Also, verify the track titles are in the following format: Artist / Song Title. If you are ripping a Soundtrack CD comprised of various artists, be sure to select the Various Artists check box in the main EAC window and enter the CD Title in the following format: Soundtrack - Film Title.

Scratched CDs:  EAC is the best program out there for correctly reading scratched CDs.  You will find, however, that some CD-ROM drives are better at dealing with poor CDs than others.  You may also find that some CDs are so damaged that they can not be read.  A CD that has a scratch on the top "printed" side, for example, is not readable because the actual data has been scratched.  

Luckily, most CDs that are scratched simply have minimal damage to the underside layer of the disc.  In these situations, I full heartedly recommend the SkipDr. scatch removal system.  This product is a life saver.  At around $30 dollars, it has more than earned its keep.  Personally, I have saved at least 25 CDs with this product.  It works by polishing the surface of the disc, layer by layer, until the scratch is removed.   It works for DVDs too.
