My Empeg Stuff

I've written some programs for the empeg car mp3 player. All are available under the GNU Public License free for personal use, and with complete source. Feel free to use these to help you learn how to get started writing apps for the Empeg.

Note: These programs require Hijack Empeg kernels.

  • emphatic - a lyrics viewer
  • emptriv - a text-based trivia game
  • empwake - an alarm clock
  • empan - an image panner/viewer
