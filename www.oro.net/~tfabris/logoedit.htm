Empeg Logo Editor is a program to import, export, edit, and upload the startup images on the empeg car digital music player.

It allows direct import/export of Windows Bitmap files, and will copy and paste from the Windows Clipboard to support other image types.

Get Empeg Logo Editor Now.
Size=111k. Time to download at 28.8=Less than a minute.

To install it, simply download it from this site, unzip it to any directory you choose, and run LOGOEDIT.EXE.

Note: If you get an error message saying that a DLL file is missing or "out of date", then you also need to install the Visual Basic 6 Runtime Files. You might already have these files, so try running Logo Editor first, before wasting the time to download these files.

This software has the following system requirements:

  • Windows 95, 98, Windows NT, or Windows 2000.

  • The Visual Basic 6 Runtime Files.

  • A video card capable of 24-bit color.

  • An empeg car digital music player.

  • A working COM port (for uploading the finished logos to the empeg).


For examples of other user's logo files, visit Drakino's Logo Site.
